La historia de HTML

HTML, siglas en inglés de Hyper Text Markup Language (lenguaje de marcas de hipertexto)
hace referencia al lenguaje de marcado para la elaboración de páginas web.
Es un estándar que sirve de referencia del software que conectaron la elaboración
de paginas web en sus diferentes versiones, define una estructura básica
y un código (denominado código HTML) para la definición de contenido
de una página web, como texto, imágenes, videos, juegos, entre otros.

Versiones de HTML:
  • Versión 2.0 1995
  • Versión 3.2 1997
  • Versión 4.0 1999
  • Versión 4.01 1999
  • Versión 5.0 2012

Tim Bemers-Lee (TBL) en 1991 describe 18 elementos que incluyen el diseño
inicial y relativamente simple de HTML
Trece estos elementos todavía existen en HTML 4. Berners-Lee consideraba
el HTML una ampliación de SGML,pero no fue formalmente reconocida como tal hasta
la publicación a mediados de 1993, por la IETF (en español: Grupo de Trabajo de Ingenieria de Internet),
de una primera proposición para una especificación del HTML:
El borrador expiró a los seis meses, pero fue notable por su reconocimienta de la etiqueta
propia del navegador Mosaic usada para insertar imágenes sin cambio de
linea, que reflejaba la filosofia del IETF de basar estándares en prototipos con éxito.
